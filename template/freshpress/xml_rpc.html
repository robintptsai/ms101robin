Notice:
  • This function makes you be able to manage your blog by blog publishing software (eg: Windows Live Writer). Set user name or password to empty can disable this function.
  • It support Blogger API, MetaWeblog API and some method of WordPress API as so far.
  • If your blog publishing software can't auto-detect, you can input these addresses manually: RSD address is #{{BLOG_HOME_FULL_URL}}rsd, XML-RPC API address is #{{BLOG_HOME_FULL_URL}}rpc.
