Flush cache

Flush memcache and server cache.

Generate sitemap

Generate sitemap manually. It may take a while to finish. It will be automatically generated once a day, so you should do this operation only after import many articles.

Fix datastore

Search all articles and comments, generate the missing categories, tags and users. You can use it after importing if you meet some error.

Generate historical feed

You can use it to generate historical feed and notify Google Reader to record it. You should change the URL mappings at blog.py, uncomment the line contains view.GeneratedFeedPage.

Update tags count

You can use it to update tags' articles count if incorrect.

Update articles' replies

You can use it to update articles' replies if incorrect.

Update subscribers

You can use it to update feed subscribers if incorrect.
